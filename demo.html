Redux

As our app got more complex managing the state felt messy and overwhelming. Using Redux for state management made it easier to maintain these states across our app.

As a personal goal I wanted to use this and understand it more

JSON Web Token(JWT) and bcrypt

JWT:

After logging in, each subsequent request will include the JWT, allowing the user to access routes, services, and resources that are permitted with that token.

bcrypt:

This was used to hash the password in our database so passwords werent being stored directly to our database.

Gravatar

or Globally Recognized Avatar:

This was a fairly easy way to imort profile pictures to our site. The user uploads an image and creates a profile just once, and then when you participate in any Gravatar-enabled site, your Gravatar image will automatically follow you there.

Gravatar is a free service for site owners, developers, and users. It is automatically included in every WordPress.com account and is run and supported by Automattic.
